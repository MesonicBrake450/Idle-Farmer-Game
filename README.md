# Farmer Game
    Description: 
    My project is Farmer Game, the game is a mostly UI, clicker
    and simulation game. The user can level up by doing things such
    as planting crops, selling crops, buying seeds. When you first
    boot up the game the one of only buttons you can click is the "Plant
    Potato" button. when you plant the potatoes you will get level
    points, level points are not shown to the user, but they are used
    as a universal points system which is what lets the user level up.
    At level 2 the user unlocks planting corn, level 3 unlocks wheat.
    There is a "Sell Crops" button which picks a random crop you have 
    and takes one of that crop from your storage and you gain money
    depending on the crop, $3 for potatoes, $5 for corn, and $7 for wheat.
    if the there is none of the random crop then it chooses a different one
    until and if there are no crops the button is disabled.
    Next to the "Sell Crops" button is the "Buy Seeds" button, which 
    lets you buy 5 seeds for 10 dollars. in order to plant a crop
    you need to use up 2 seeds. The user starts with a balance of $20
    and 20 seeds. In the top right corner of the screen there are four buttons
    Upgrade Storage, Hire Farmes, Upgrade Tools, and Hire Marketers.
    Upgrade Storage increases the max amount of crops you can have
    at one time, Hire Farmers plants one crops for every second, 
    Upgrade Tools gives you more plants per click, and Hire Marketers
    automatically sells one crop every second, and buys two seeds for
    $3 every two seconds. You can only upgrade storage once per level.
    You are able to Upgrade Storage throughout the entire game from level one.
    Upgrading Tools unlocks at level three, Farmers at four, and Marketers at five.
